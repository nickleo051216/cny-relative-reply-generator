## ✨ 功能特色

- 🎯 **15+ 常見親戚問題** - 什麼時候結婚、年薪多少、怎麼還不買房...
- 🔥 **4 種回嘴風格** - 機車嗆爆、犀利反擊、幽默諷刺、溫柔刀
- 🤖 **AI 智慧生成** - Claude AI 幫你想出最嗆的回覆
- 🧧 **50+ 吉祥話** - 嗆完還是要祝福，智慧配對分類
- 📱 **一鍵複製分享** - 截圖分享到 Threads / LINE

## 🚀 快速開始

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 打包部署
npm run build
```

## 🎨 技術棧

- React 18
- Vite 5
- Tailwind CSS 3
- Claude AI API
